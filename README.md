# AS 2026 Cambridge Dashboard (Zone 4)

A web-based dashboard designed for Cambridge International AS Level students in Administrative Zone 4 to track exam schedules and access revision resources.

## Features

* **Exam Timetable:** Live countdowns and schedules for subjects including Mathematics, Physics, Chemistry, Computer Science, and more.
* **Syllabus Integration:** On-page viewing of specific paper topics and syllabus headings.
* **Past Paper Archive:** Organized links to past papers and marking schemes sourced from external repositories.
* **Revision Tools:** Estimated study hour allocations, structured daily routines, and consistency strategies.
* **Sorting & Filtering:** Options to organize exams by date or subject for easier tracking.
* **Mobile Optimization:** Fully responsive design for use on mobile and desktop devices.

## Project Structure

* `index.html`: The primary dashboard containing the master timetable and study routines.
* `subject-resources.html`: The resource hub for subject-specific past paper links.
